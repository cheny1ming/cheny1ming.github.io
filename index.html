MLLM + Geolocation

基于 MLLM 的智能地理位置识别系统 Intelligent Geolocation Recognition System Based on MLLM

针对即时零售场景下的识别痛点，基于 Qwen2.5-VL 实现地址自动提取与精准检索，显著提升配送效率。 Addressed pain points in instant retail scenarios by implementing automated address extraction and precision retrieval via Qwen2.5-VL, enhancing delivery efficiency.

  • • 采用 LoRA 与全参数微调 (DeepSpeed)，实现 OCR 准确率 99% 与分类准确率 98%。 • Leveraged LoRA & Full-parameter Fine-tuning (DeepSpeed) to achieve 99% OCR accuracy and 98% classification accuracy.
  • • 融合 高德 API 与多语义搜索策略，将 Top-1 检索准确率从 91% 提升至 96%。 • Integrated Amap API with multi-semantic search, boosting Top-1 retrieval accuracy from 91% to 96%.
  • • 手动构建 2% 噪声数据集，有效增强模型泛化能力并降低异常场景误判率。 • Manually curated a 2% noise dataset, enhancing generalization and reducing misjudgment in edge cases.

MLLM + Safety Moderation

基于 MLLM 的内容安全审核平台 Content Safety Moderation Platform Powered by MLLM

利用多模态大模型替代传统人工模式，实现跨场景内容的自动化违规识别与归因，助力企业降本增效。 Substituted traditional manual audits with MLLMs for automated multi-scenario violation detection and reasoning, reducing operational costs.

  • • 强化学习落地： 采用 GRPO 算法 构建混合奖励模型，将审核召回率从 70% 提升至 85%。 • RL Implementation: Utilized the GRPO algorithm to build a hybrid reward model, increasing recall from 70% to 85%.
  • • 冷启动与数据工程： 基于 GPT-4o 生成审核原因标签，构建 1.5W+ 均衡数据集，解决样本不平衡问题。 • Cold Start & Data Engineering: Generated reasoning labels via GPT-4o and built a balanced 15k+ dataset to mitigate sample imbalance.
  • • 性能优化： 针对业务需求调整 Token 预测概率，实现了 98% 的极高审核精确率。 • Optimization: Adjusted token prediction probabilities to reach a 98% precision rate for mission-critical automation.
